# Export instructions

After you finalize a workflow in the n8n UI:

1. open the workflow
2. use the export option in n8n
3. save the exported JSON into `workflows/exports/`
4. use a stable filename matching the workflow number and name

Recommended filenames:

- `01_weekly_strategy.json`
- `02_asset_intake.json`
- `03_content_generation.json`
- `04_send_drafts_for_approval.json`
- `05_populate_review_queue.json`
- `06_ready_to_schedule.json`
